HTML / CSS / JavaScript Portfolio Bundle

This bundle contains multiple portfolio website templates built using
pure HTML, CSS, and vanilla JavaScript.

------------------------------------------------

How to Use
1. Open any Portfolio folder (Portfolio-01, Portfolio-02, etc.)
2. Open index.html in your browser
3. To edit content:
   - Open index.html in a code editor
   - Change text, links, and sections directly
4. To change images:
   - Replace files inside the images/ folder
   - Keep the same file names

------------------------------------------------

Folder Structure (per portfolio)
- index.html   -> Main entry file
- css/         -> Stylesheets
- js/          -> JavaScript files
- images/      -> Images and icons

------------------------------------------------

Tech Stack
- HTML
- CSS
- Vanilla JavaScript
(No frameworks, no build tools)

------------------------------------------------

Who This Is For
- Students
- Beginners learning frontend
- Anyone who needs a quick portfolio layout

------------------------------------------------

Notes
- These templates are for personal or learning use
- You may customize them freely
